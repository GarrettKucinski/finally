## Integration Issues (Cross-Phase)

### 1. CHAT-05: Trade Auto-Execution → User Confirmation (Deliberate Divergence)

**Requirement:** "Trades in LLM response are auto-executed through the same execute_trade path as manual trades"
**Actual:** Trades are returned as `proposed_trades` in API response. Frontend renders confirm/dismiss cards. User must click "Confirm" to execute.
**Evidence:** `backend/app/services/chat.py:222` docstring: "Trades are NOT executed here — they are returned as proposed_trades"; system prompt instructs "trades are never auto-executed"; test `test_trades_never_executed` exists.
**Impact:** This is a deliberate product decision that improves UX safety. The end-to-end wiring (propose → render → confirm → execute) is correct. Watchlist changes (CHAT-06) ARE auto-executed as specified.
**Severity:** Info — intentional divergence, not a defect

### 2. UI-02: CSS Token Naming Mismatch (Rendering Defect)

**Defined in globals.css:** `--color-secondary-purple: #753991`
**Referenced by components:** `--color-accent-purple` (undefined)
**Affected elements:**
- `ChatPanel.tsx:105` — Send button background (transparent instead of purple)
- `ChatMessage.tsx:32` — User message bubble background
- `Dashboard.tsx:130` — Floating AI toggle button (`bg-accent-purple` Tailwind class)
**Impact:** Three elements render without purple background. Buttons still function.
**Severity:** Minor — visual defect only

### 3. VIZ-01: Change% Baseline (Behavioral Divergence)

**Decision Q4:** "Use seed price as the baseline. Label as 'Change' (not 'Daily change')."
**Actual:** `PriceUpdate.change_percent` computes `(price - previous_price) / previous_price * 100` (tick-to-tick), which is near-zero and resets on each SSE event.
**Impact:** Change% column in watchlist has minimal informational value compared to seed-baseline %.
**Severity:** Minor — display logic difference

### 4. VIZ-09 / OPS-06: Floating Button Accessible Name Mismatch

**Component:** `Dashboard.tsx` floating button has `title="Open AI Chat"` with text content as SVG icon
**E2E test:** `page.getByRole('button', { name: 'AI' })` — Playwright resolves accessible name from `title` → "Open AI Chat", not "AI"
**Impact:** Chat collapse/reopen E2E test will fail (timeout). 16/17 other tests unaffected.
**Severity:** Minor — one E2E test selector needs updating

### 5. Minor: WatchlistRow stopPropagation

**File:** `WatchlistPanel.tsx:68` — Remove button onClick doesn't call `e.stopPropagation()`, causing simultaneous ticker selection and removal.
**Impact:** Brief visual jitter when removing a ticker. Non-blocking.
**Severity:** Cosmetic
