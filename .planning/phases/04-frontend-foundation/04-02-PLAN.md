**Step 1: Create number formatting utilities** `frontend/src/lib/format.ts`:
```typescript
export function formatCurrency(value: number): string {
  return new Intl.NumberFormat('en-US', {
    style: 'currency',
    currency: 'USD',
    minimumFractionDigits: 2,
    maximumFractionDigits: 2,
  }).format(value)
}

export function formatPercent(value: number): string {
  const sign = value >= 0 ? '+' : ''
  return `${sign}${value.toFixed(2)}%`
}

export function formatQuantity(value: number): string {
  return value % 1 === 0 ? value.toString() : value.toFixed(4)
}
```
- `formatCurrency` returns `$1,234.56` format.
- `formatPercent` returns `+2.45%` or `-1.30%` (always shows sign).
- `formatQuantity` returns integers without decimals, fractionals with up to 4 decimal places.

**Step 2: Create PriceFlash component** `frontend/src/components/ui/PriceFlash.tsx`:
This is the reusable component for price cells that flash green/red on change (UI-05).

```typescript
'use client'
import { useEffect, useRef, useState } from 'react'
import { formatCurrency } from '@/lib/format'

interface PriceFlashProps {
  price: number
  direction: 'up' | 'down' | 'flat' | string
  className?: string
}

export function PriceFlash({ price, direction, className = '' }: PriceFlashProps) {
  const [flash, setFlash] = useState<'up' | 'down' | null>(null)
  const prevPrice = useRef(price)
  const timerRef = useRef<ReturnType<typeof setTimeout> | null>(null)

  useEffect(() => {
    if (price !== prevPrice.current) {
      // Clear any existing timer to handle rapid updates
      if (timerRef.current) clearTimeout(timerRef.current)

      setFlash(direction === 'up' ? 'up' : direction === 'down' ? 'down' : null)
      prevPrice.current = price

      timerRef.current = setTimeout(() => {
        setFlash(null)
        timerRef.current = null
      }, 500)
    }

    return () => {
      if (timerRef.current) clearTimeout(timerRef.current)
    }
  }, [price, direction])

  return (
    <span
      className={`
        inline-block px-1 rounded transition-colors duration-500 ease-out
        ${flash === 'up' ? 'bg-price-up/30' : ''}
        ${flash === 'down' ? 'bg-price-down/30' : ''}
        ${className}
      `.trim()}
    >
      {formatCurrency(price)}
    </span>
  )
}
```
Key details:
- `'use client'` directive required.
- Uses `useRef` for both previous price and timer, preventing stale closures.
- Clears previous timer on rapid updates (prevents stuck flash from Pitfall 3).
- CSS `transition-colors duration-500 ease-out` handles the visual fade.
- `bg-price-up/30` and `bg-price-down/30` use Tailwind opacity modifier with the custom theme colors.
- Cleanup in useEffect return prevents memory leaks.
