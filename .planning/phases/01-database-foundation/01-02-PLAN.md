    **2. Create `backend/app/routes/health.py`:**

    ```python
    from fastapi import APIRouter, Request
    from fastapi.responses import JSONResponse

    router = APIRouter(tags=["system"])

    @router.get("/api/health")
    async def health(request: Request):
        pool = request.app.state.db_pool
        try:
            async with pool.acquire() as conn:
                await conn.fetchval("SELECT 1")
            return {"status": "healthy", "database": "connected"}
        except Exception:
            return JSONResponse(
                status_code=503,
                content={"status": "unhealthy", "database": "disconnected"},
            )
    ```

    Key details:
    - Access pool via `request.app.state.db_pool` (standard FastAPI pattern, no global state)
    - Catch broad Exception to handle any pool/connection error (connection refused, timeout, pool exhausted)
    - Return 200 for healthy, 503 for unhealthy (standard for health checks)
    - Do NOT use the error response format from PLAN.md Section 8 (`{"error": ..., "detail": ...}`) -- health checks have their own convention

    **3. Create `backend/tests/test_health.py`:**

    Use `httpx.AsyncClient` with the FastAPI app's `ASGITransport` to test the endpoint. Since the full lifespan requires a real database, mock `app.state.db_pool` to isolate the health route logic:

    For the healthy test:
    - Create a mock pool with `acquire()` returning an async context manager whose `fetchval()` returns 1
    - Override `app.state.db_pool` with the mock
    - Assert response status 200 and body matches expected JSON

    For the unhealthy test:
    - Create a mock pool whose `acquire()` raises `Exception("Connection refused")`
    - Override `app.state.db_pool` with the mock

    Use `from app.main import app` and mock app.state. Use `httpx.ASGITransport(app=app)` with `httpx.AsyncClient` for testing (avoids needing TestClient or running the real lifespan).

    Important: Since `app.main` imports Settings which may try to read env vars, either:
