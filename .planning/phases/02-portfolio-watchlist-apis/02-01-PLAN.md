<objective>
Implement the portfolio data models, trade execution service, and REST API endpoints for viewing portfolio state, executing trades, and querying portfolio history.

Purpose: This is the core trading engine of FinAlly. Trade execution with atomic transactions is the most complex business logic in the system -- it validates cash/shares, updates positions (with weighted average cost), modifies cash balance, logs trades, and records snapshots, all within a single DB transaction. The portfolio query enriches DB positions with live prices from the market data cache to compute unrealized P&L.

Output: Pydantic models, portfolio service layer, three API endpoints, and comprehensive tests.
</objective>

       - Use `async with pool.acquire() as conn:` then `async with conn.transaction():`

    ```python
    from fastapi import APIRouter, Request
    from fastapi.responses import JSONResponse
    from app.models.portfolio import TradeRequest, TradeResponse, PortfolioResponse
    from app.models.common import ErrorResponse
    from app.services.portfolio import execute_trade, get_portfolio, record_snapshot

    router = APIRouter(tags=["portfolio"])

    @router.get("/api/portfolio")
    async def get_portfolio_endpoint(request: Request):
        pool = request.app.state.db_pool
        price_cache = request.app.state.price_cache
        result = await get_portfolio(pool, price_cache)
        return result

    @router.post("/api/portfolio/trade")
    async def trade_endpoint(request: Request, body: TradeRequest):
        pool = request.app.state.db_pool
        price_cache = request.app.state.price_cache
        try:
            result = await execute_trade(pool, price_cache, body.ticker, body.side, body.quantity)
            # Record snapshot after successful trade (PORT-09)
            await record_snapshot(pool, price_cache)
            return result
        except ValueError as e:
            return JSONResponse(
                status_code=400,
                content=ErrorResponse(error="Trade failed", detail=str(e)).model_dump(),
            )
    ```

    ```python
    router = APIRouter(tags=["portfolio"])

    @router.get("/api/portfolio/history")
    async def get_portfolio_history(request: Request):
        pool = request.app.state.db_pool
        DEFAULT_USER_ID = "00000000-0000-0000-0000-000000000001"
        async with pool.acquire() as conn:
            rows = await conn.fetch(
                "SELECT total_value, recorded_at FROM portfolio_snapshots WHERE user_id = $1 ORDER BY recorded_at",
                DEFAULT_USER_ID,
            )
        return [{"total_value": round(row["total_value"], 2), "recorded_at": row["recorded_at"].isoformat()} for row in rows]
    ```
